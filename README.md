# DSA Coding Challenge Application

This project is an **online coding challenge platform** developed specifically for a club event to conduct LeetCode-style DSA contest.  

It serves as a **demonstration of system design and implementation**, rather than a reusable or production-ready platform.

---

## 🚀 Project Overview

The application was built to simulate a real-world coding contest environment with:

- Problem solving interface  
- Automated code evaluation  
- Timed contest experience  
- Performance-based ranking  

It was successfully used to organize and manage a structured DSA event.

---

## ✨ Key Features

- 🌐 **Web-Based Platform** (requires internet connection)  
- 🧑‍💻 **Supports Python & Java submissions**  
- ⚡ **Automated Judging System** for instant feedback  
- ⏱️ **Contest Timer** with fixed duration  
- 🏆 **Performance Classification** (Gold / Silver / Bronze)  
- 📊 **Leaderboard System** for ranking participants  
- 🧩 **Predefined DSA Problem Set** (Easy & Medium levels)  

---

## 🛠️ Tech Stack

- **Frontend**: HTML, CSS, JavaScript  
- **Backend**: Flask (Python)  
- **Database**: SQLite  
- **Code Execution**: Subprocess-based sandboxing  

---

## 🎯 Purpose

This project was created to:

- Conduct a **club-level coding contest**
- Provide participants with a **real contest experience**
- Demonstrate concepts like:
  - Backend system design  
  - Code execution engines  
  - Automated judging logic  
  - Full-stack integration  

---

## ⚙️ How It Works (High-Level)

1. Users register and start the contest  
2. Problems are fetched from the backend  
3. Code submissions are sent to the server  
4. Server executes code against test cases  
5. Results and scores are updated dynamically
